ObjectFactory is an abstraction from XWork2 responsible for all the creation of XWork2 related objects. Its how XWork2 integrate with IoC containers like Spring, Pico, Plexus etc.

Customization

To customize an ObjectFactory, simply have a static block that set a cutom instance of ObjectFactory as follows :-

Custom ObjectFactory must extends ObjectFactory or any of its child.

static {
   ObjectFactory.setObjectFactory(new CustomObjectFactory(....));
 }
