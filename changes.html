This is the XWork changelog. This is a high-level list of changes and bug fixes. A more exhaustive list of issues is maintained in the issue tracker.

1.0.1 - Mar 21, 2004

    Key Bugfixes:
    • XW-156 - Added actionMessages support
    • XW-167 - XWorkBasicConverter conversion from Date to String is not localized
    • Exceptions occuring during OgnlValueStack.findValue() are now logged correctly
    • Huge performance improvement when Ognl 2.6.4 is used (previous version used a modified form of 2.6.3)
    New Features:
    • XW-162, XW-161 - Replaceable ObjectFactory for creating framework objects to allow easier integration with IoC containers
    • XW-160 - Add infinite recursion detection to the ChainingInterceptor

1.0 - Feb 9, 2004

    Initial release of XWork.
