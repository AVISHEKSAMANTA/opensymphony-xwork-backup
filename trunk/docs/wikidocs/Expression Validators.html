The Expression validator extends the ValidatorSupport base class and has one parameter which should be set using:

<param name="expression">any valid OGNL expression which returns a boolean</param>.

The OGNL expression will be evaluated against the ValueStack, allowing you to use any properties of the Action in comparisons, etc. This allows for very powerful cross-property validation expressions. Here is a simple example of the ExpressionValidator in a configuration:

<validator type="expression">
        <param name="expression">foo > bar</param>
        <message default="Foo must be greater than Bar. Foo = ${foo}, Bar = ${bar}."/>
</validator>
